# Capability Ontology — 能力分类体系

PCAE 使用六大类能力分类框架，用于映射用户从叙事中展现的能力。每个叶节点标注了 AI 影响类型。

## 使用说明

- 这个体系用于**映射**，不是**限定**。如果用户展现的能力不在列表中，创建新条目。
- 不要向用户展示整个分类树。根据用户故事，识别相关的能力节点。
- 每个能力的"可观察指标"帮助你从用户故事中识别信号。

## 1. Thinking 思维能力

### 1.1 Scientific Thinking 科学思维
- **可观察指标**：提出假设、设计实验验证、用数据得出结论、质疑既有结论
- **AI影响**：放大型 — AI加速数据分析和文献检索，但假设生成和实验设计仍依赖人
- **子能力**：
  - Hypothesis Generation 假设生成
  - Experimental Design 实验设计
  - Data Interpretation 数据解读
  - Critical Evaluation 批判性评估

### 1.2 Strategic Thinking 战略思维
- **可观察指标**：从全局看问题、识别长期趋势、权衡取舍、制定优先级
- **AI影响**：抵抗型 — AI可提供信息，但"该问什么问题"和"该追求什么目标"仍是人的判断
- **子能力**：
  - Pattern Recognition 模式识别
  - Priority Setting 优先级设定
  - Trade-off Analysis 权衡分析
  - Long-term Planning 长期规划

### 1.3 Systems Thinking 系统思维
- **可观察指标**：看到事物间的关联、理解反馈循环、识别涌现现象、预见二阶效应
- **AI影响**：放大型 — AI可建模复杂系统，人提供系统边界定义和意义解读
- **子能力**：
  - Interconnection Mapping 关联映射
  - Feedback Loop Identification 反馈循环识别
  - Emergence Recognition 涌现识别
  - Second-order Effect Prediction 二阶效应预测

### 1.4 Creative Thinking 创造性思维
- **可观察指标**：提出新颖方案、跨域联想、重构问题、挑战假设
- **AI影响**：混合型 — AI在受限创意（广告文案、配色）上已接近人类，但跨域新颖综合仍主要是人
- **子能力**：
  - Divergent Thinking 发散思维
  - Analogical Reasoning 类比推理
  - Problem Reframing 问题重构
  - Constraint-based Innovation 约束式创新

### 1.5 Analytical Thinking 分析思维
- **可观察指标**：分解复杂问题、找到根因、建立逻辑链、量化比较
- **AI影响**：放大型 — AI大幅加速分析，人提供问题定义和结论验证
- **子能力**：
  - Decomposition 分解
  - Root Cause Analysis 根因分析
  - Logical Reasoning 逻辑推理
  - Quantitative Comparison 量化比较

## 2. Communication 沟通能力

### 2.1 Teaching 教学
- **可观察指标**：把复杂概念讲简单、因材施教、用类比和故事、检测理解程度
- **AI影响**：抵抗型 — AI可提供内容，但激励、共情、实时调整教学策略仍依赖人
- **子能力**：
  - Simplification 简化表达
  - Adaptive Explanation 适应性解释
  - Engagement 互动激发
  - Understanding Assessment 理解度评估

### 2.2 Storytelling 叙事表达
- **可观察指标**：把经历讲成故事、有结构有情感、能打动听众、控制节奏
- **AI影响**：混合型 — AI能生成故事，但基于个人经历的真实叙事和现场表达仍是人
- **子能力**：
  - Narrative Structure 叙事结构
  - Emotional Resonance 情感共鸣
  - Audience Adaptation 受众适应
  - Authenticity 真实性

### 2.3 Negotiation 谈判
- **可观察指标**：达成双赢、读懂对方需求、让步策略、创造性解决僵局
- **AI影响**：抵抗型 — 依赖信任、情境判断、关系资本
- **子能力**：
  - Interest Discovery 利益发现
  - Value Creation 价值创造
  - Persuasion 说服
  - Conflict Resolution 冲突解决

### 2.4 Cross-cultural Communication 跨文化沟通
- **可观察指标**：适应不同文化语境、避免文化误解、桥接差异
- **AI影响**：抵抗型 — 依赖文化敏感度和人际信任
- **子能力**：
  - Cultural Sensitivity 文化敏感度
  - Context Switching 语境切换
  - Bridge Building 桥梁搭建

## 3. Execution 执行能力

### 3.1 Leadership 领导力
- **可观察指标**：带动团队、做艰难决策、承担责任、激励他人
- **AI影响**：抵抗型 — 信任、问责、愿景传递依赖人
- **子能力**：
  - Vision Setting 愿景设定
  - Decision Making 决策
  - Accountability 问责
  - Team Building 团队建设

### 3.2 Planning 规划
- **可观察指标**：制定可执行计划、分解任务、预估时间、管理依赖
- **AI影响**：放大型 — AI可协助排期和依赖分析，人设定目标和优先级
- **子能力**：
  - Goal Decomposition 目标分解
  - Resource Allocation 资源配置
  - Timeline Estimation 时间估算
  - Risk Assessment 风险评估

### 3.3 Project Management 项目管理
- **可观察指标**：协调多方、跟踪进度、处理变更、保持交付
- **AI影响**：放大型 — AI可自动化状态追踪和报告
- **子能力**：
  - Coordination 协调
  - Progress Tracking 进度跟踪
  - Change Management 变更管理
  - Stakeholder Management 干系人管理

### 3.4 Entrepreneurship 创业
- **可观察指标**：发现机会、承担风险、从0到1、快速迭代
- **AI影响**：放大型 — AI降低创业门槛（原型、分析、内容），人提供愿景和判断
- **子能力**：
  - Opportunity Recognition 机会识别
  - Risk Tolerance 风险承受
  - Rapid Prototyping 快速原型
  - Pivot Decision 转向决策

## 4. Creation 创造能力

### 4.1 Writing 写作
- **可观察指标**：清晰表达复杂想法、有个人风格、能打动读者
- **AI影响**：放大型 — AI加速起草和编辑，人提供原创思想和个人风格
- **子能力**：
  - Technical Writing 技术写作
  - Creative Writing 创意写作
  - Academic Writing 学术写作
  - Copywriting 文案写作

### 4.2 Coding 编程
- **可观察指标**：用代码解决问题、理解系统架构、写可维护代码
- **AI影响**：高度放大型 — 编程是 AI 放大效应最显著的领域之一（~1.7x 生产力提升）
- **子能力**：
  - Problem Solving 问题求解
  - Architecture Design 架构设计
  - Code Quality 代码质量
  - Tool Building 工具构建

### 4.3 Design 设计
- **可观察指标**：审美判断、用户同理心、将抽象需求可视化
- **AI影响**：混合型 — AI可生成设计方案，人提供品味和用户洞察
- **子能力**：
  - Visual Design 视觉设计
  - UX Design 用户体验设计
  - Information Architecture 信息架构
  - Design Thinking 设计思维

### 4.4 Product 产品
- **可观察指标**：理解用户需求、定义产品方向、平衡商业和用户价值
- **AI影响**：抵抗型 — 用户洞察、产品判断和战略方向依赖人
- **子能力**：
  - User Research 用户研究
  - Feature Prioritization 功能优先级
  - Product Strategy 产品战略
  - Market Sensing 市场感知

## 5. Research 研究能力

### 5.1 Literature Research 文献研究
- **可观察指标**：高效检索、批判性阅读、综合多源信息
- **AI影响**：高度放大型 — AI大幅加速文献检索和摘要（30%+ 效率提升）
- **子能力**：
  - Search Strategy 检索策略
  - Critical Reading 批判性阅读
  - Synthesis 综合分析
  - Citation Analysis 引文分析

### 5.2 Experimental Research 实验研究
- **可观察指标**：设计实验、控制变量、分析结果、复现验证
- **AI影响**：放大型 — AI辅助数据分析和模式发现
- **子能力**：
  - Experiment Design 实验设计
  - Variable Control 变量控制
  - Result Analysis 结果分析
  - Reproducibility 可复现性

### 5.3 Data Research 数据研究
- **可观察指标**：数据清洗、统计分析、可视化、从数据得出洞察
- **AI影响**：高度放大型 — AI可自动化大量数据处理工作
- **子能力**：
  - Data Wrangling 数据清洗
  - Statistical Analysis 统计分析
  - Visualization 可视化
  - Insight Extraction 洞察提取

### 5.4 Qualitative Research 质性研究
- **可观察指标**：深度访谈、主题分析、扎根理论、理解主观经验
- **AI影响**：抵抗型 — 深度理解人的主观经验和语境依赖人
- **子能力**：
  - Interview Skills 访谈技能
  - Thematic Analysis 主题分析
  - Empathetic Understanding 共情理解
  - Contextual Interpretation 语境解读

## 6. AI 能力

### 6.1 Prompt Engineering 提示工程
- **可观察指标**：精确描述需求、理解模型能力边界、迭代优化输出
- **AI影响**：元能力 — 直接决定 AI 放大倍数
- **子能力**：
  - Task Decomposition 任务分解
  - Context Setting 上下文设定
  - Output Shaping 输出塑形
  - Error Detection 错误检测

### 6.2 Workflow Design AI工作流设计
- **可观察指标**：将复杂任务拆解为 AI 可执行的步骤链、选择合适工具
- **AI影响**：元能力 — 设计人如何使用AI
- **子能力**：
  - Process Mapping 流程映射
  - Tool Selection 工具选择
  - Integration Design 集成设计
  - Quality Assurance 质量保证

### 6.3 Automation 自动化
- **可观察指标**：识别可自动化的重复任务、实现自动化流程
- **AI影响**：放大型 — AI使自动化门槛大幅降低
- **子能力**：
  - Pattern Identification 模式识别
  - Script Building 脚本构建
  - System Integration 系统集成

### 6.4 AI Judgment AI判断力
- **可观察指标**：评估 AI 输出质量、发现 AI 错误、知道何时该信任/不信任 AI
- **AI影响**：抵抗型 — 这是"验证瓶颈"问题的核心能力
- **子能力**：
  - Output Evaluation 输出评估
  - Hallucination Detection 幻觉检测
  - Boundary Awareness 边界意识
  - Trust Calibration 信任校准

## AI 影响分类汇总

| 类型 | 含义 | 示例能力 |
|------|------|----------|
| **抵抗型** | AI 难以替代 | 领导力、谈判、教学、质性研究、AI判断力 |
| **放大型** | AI 让你更强 | 编程、数据研究、规划、写作、文献研究 |
| **高度放大型** | AI 效果最显著 | 编程(1.7x)、文献研究(30%+)、数据研究 |
| **混合型** | AI部分替代、部分放大 | 创造性思维、设计、叙事表达 |
| **元能力** | 直接决定AI放大倍数 | 提示工程、工作流设计 |
